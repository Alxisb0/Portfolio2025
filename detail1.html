Interface administrateur pour mettre à jour les données d'un produit

Cette page fait partie du site e-commerce que nous avons développé en Python avec Flask dans le cadre de la SAE 2.04.

Ici, l’administrateur peut gérer les boissons et leurs déclinaisons : modifier le nom, changer le prix, gérer le stock, ajouter ou supprimer des déclinaisons selon le volume, etc. On trouve aussi un bouton pour se déconnecter, ainsi qu’un message qui confirme quand une modification a été prise en compte.

Pour que seul un administrateur puisse accéder à cette page, j’ai mis en place un système de gestion des sessions sous Flask qui vérifie le rôle à chaque chargement.

Pour modifier les données, j’utilise des requêtes SQL UPDATE qui mettent à jour précisément les informations dans la base, par exemple :
UPDATE declinaison SET stock = ... WHERE id_declinaison = ....

La récupération des données se fait via les méthodes GET et POST de Flask : l’identifiant de la déclinaison est pris dans l’URL grâce à request.args.get(), et les nouvelles valeurs du formulaire avec request.form.get().

Ce projet m’a permis de relier un site web fonctionnel à une base de données, de sécuriser les pages sensibles, et de manipuler les données utilisateur avec Flask, ce qui représente une application concrète de mes compétences en développement web Python.
